## Prerequisite

- Linux/Unix OS
- docker & docker-compose

## Getting Started

```
git init
```

```
git clone https://github.com/kmlee78/jjabphasquare.git
```

## Development Process

Move to the project directory and run the codes.

- build development server

```
make dev-up
```

- terminate development server

```
make dev-down
```

- execute temporary shell

```
make dev-shell
```
